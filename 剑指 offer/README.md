1. 门槛较高，要有很好的数学基础和计算机专业其他课程基础
2. 数据结构与算法本身就非常有用，在工作中会产生很大的价值
3. 说明思维能力很好
4. 说明综合素质较好，有恒心，有毅力，可以持之以恒，不怕困难

2. 可以先找一个具体的例子分析  
3. 把大问题分解成小问题
4. 逆向思维，从初始条件正着想，从想要的结果反着想  
5. 考虑边界条件
