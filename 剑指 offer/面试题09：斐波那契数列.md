
题目描述  
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0，第1项是1）。  


方法 1：  
使用递归实现  
```python
class Solution:
    def Fibonacci(self, n):
        if n == 0 or n == 1:
            return n
        else:
            return self.Fibonacci(n - 1) + self.Fibonacci(n - 2)


s = Solution()
s.Fibonacci(4)
```

复杂度过高，为指数阶  

方法 2：
```python
class Solution:
    def Fibonacci(self, n):
        if n == 0 or n == 1:
            return n
        else:
            fib0 = 0
            fib1 = 1
            for _ in range(2, n + 1):
                fib0, fib1 = fib1, fib0 + fib1
            return fib1


s = Solution()
s.Fibonacci(4)            
```
