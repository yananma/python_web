
题目描述  
输入一个链表，输出该链表中倒数第k个结点。  
如果该链表长度小于k，请返回空。  

解题思路  
求单向链表中倒数第 k 个数，若只循环一次，可以用两个指针，第一个指针先往前走 k-1 步，然后从第 k 步开始第二个指针指向头结点然后两个指针一起遍历，当地一个指针指向尾节点的时候, 第二个指针正好指向倒数第 k 个结点  
推广: 寻找中间节点，两个指针一起，第一个指针每次走两步，第二个指针每次走一步，快指针指到尾部,慢指针正好指到中间  

```python 
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
#
class Solution:
    def FindKthToTail(self , pHead , k):
        if not pHead or k <= 0:
            return None 
        
        fast = pHead 
        slow = pHead 
        for _ in range(k):    # 第一个指针先走 k 步
            if fast:    
                fast = fast.next 
            else:             # 如果 k 大于链表长度，返回空
                return None 
            
        while fast:
            fast = fast.next 
            slow = slow.next 
        return slow 
```
