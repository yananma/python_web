给定一个二叉树，返回该二叉树层序遍历的结果，（从左到右，一层一层地遍历）  

```python
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def levelOrder(self, root):
        if not root:
            return []
        queue = [root, ]
        result = []
        while queue:
            res = []
            for i in range(len(queue)):
                temp = queue.pop(0)
                res.append(temp.val)
                if temp.left:
                    queue.append(temp.left)
                if temp.right:
                    queue.append(temp.right)
            result.append(res)
        return result
```
