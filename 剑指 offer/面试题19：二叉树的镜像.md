
题目描述
操作给定的二叉树，将其变换为源二叉树的镜像。

    比如：    源二叉树 
                8
               /  \
              6   10
             / \  / \
            5  7 9 11
            
            镜像二叉树
                8
               /  \
              10   6
             / \  / \
            11 9 7  5

```python 
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    def Mirror(self , pRoot):
        if not pRoot:
            return None 
        pRoot.left, pRoot.right = pRoot.right, pRoot.left 
        self.Mirror(pRoot.left) 
        self.Mirror(pRoot.right) 
        return pRoot  
```
