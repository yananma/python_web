上面只能 print，不能修改，如果修改就要使用 nonlocal  

#### global 

```python
g = 10

def my_function():
    global g
    while True:
        print(g)
        g -= 1
        if g < 0:
            break

my_function()
```

#### nonlocal  

```python 
def outer():
    num = 100
    def inner():
        nonlocal num
        num = num + 1
        print(num)
    inner()

outer()
```
