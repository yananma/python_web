#### 字符串  

对数据进行分类大有裨益。  

字符串看似简单，但是能够以很多不同的方式使用。  

字符串就是一系列字符。在 Python 中，用引号括起来的都是字符串，其中引号可以是单引号，也可以是双引号。这种灵活性可以让我们能够在字符串中包含引号和撇号。  

字符串常用方法：title()、upper()、lower()、

字符串方法看源码和注释。
